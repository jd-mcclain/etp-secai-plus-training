## 📋 What Is the CompTIA SecAI+ Exam?

The **CompTIA SecAI+ (CY0-001)** certification validates the skills needed to:
- Secure AI systems, models, data pipelines, and APIs
- Identify and defend against AI-driven and AI-targeted threats
- Apply AI governance, risk, and compliance frameworks
- Implement responsible AI operations and oversight controls

**Exam Domains:**
| Domain | Weight |
|--------|--------|
| 1. Basic AI Concepts Related to Cybersecurity | ~15% |
| 2. Securing AI Systems | ~25% |
| 3. AI-Assisted Security | ~20% |
| 4. AI Governance, Risk, and Compliance | ~20% |
| 5. AI-Driven and AI-Targeted Threats | ~20% |
